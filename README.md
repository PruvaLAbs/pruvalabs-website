# PruvaLAbs. Website

PruvaLAbs. için Vite + React ile geliştirilmiş, modern, premium ve responsive tek sayfalık web sitesi.

## Teknoloji

- Vite
- React

## Projeyi Çalıştırma

1. Bağımlılıkları yükleyin:

```bash
npm install
```

2. Geliştirme sunucusunu başlatın:

```bash
npm run dev
```

3. Tarayıcıda açın:

`http://localhost:5173`

## Production Build

```bash
npm run build
npm run preview
```
